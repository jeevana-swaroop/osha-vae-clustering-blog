# OSHA VAE Clustering Blog

Static GitHub Pages blog for the OSHA occupational injury pattern discovery project.

## Files

- `index.html` — main blog page
- `style.css` — modern responsive styling
- `script.js` — cluster cards, tables, mobile nav, scroll animation
- `assets/vae-clusters.png` — VAE cluster plot
- `assets/kmeans-clusters.png` — K-Means cluster plot

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose `main` branch and `/root` folder.
6. Save and open the generated GitHub Pages URL.
